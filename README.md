This project demonstrates how to build a simple conversational chatbot using LangChain and Streamlit, with support for:

- Local LLMs via Ollama
- Cloud-based LLMs via OpenAI

The chatbot takes user input from a Streamlit UI and generates responses using a structured LangChain pipeline.

FEATURES:
Interactive Streamlit web interface
Uses LangChain Prompt Templates
Supports multiple LLM backends
Clean and minimal code structure
Easy to switch between Ollama and OpenAI

TECH STACK:
Python 3.9+
Streamlit
LangChain
Ollama (for local models)
OpenAI API (for cloud models)
dotenv (for environment variables)
